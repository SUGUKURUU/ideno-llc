### Next.js実装ガイド

#### ブログ機能の追加方法（推奨：microCMS or MDXローカルファイル）

**オプションA：ローカルMDX（現在のcontentフォルダを活用）**
```
# インストール
npm install next-mdx-remote gray-matter

# ファイル構成
content/
  why-sme-should-outsource-sns/
    index.md  ← フロントマター + 本文
app/
  blog/
    [category]/
      [slug]/
        page.tsx  ← 記事詳細ページ
    page.tsx      ← 記事一覧
```

**オプションB：microCMS（ノーコード更新が必要な場合）**
- microCMSでブログAPIを作成
- `npm install microcms-js-sdk`
- ISR（Incremental Static Regeneration）で配信

#### SEOに必要なNext.js設定

```typescript
// app/blog/[category]/[slug]/page.tsx
export async function generateMetadata({ params }) {
  const article = getArticleBySlug(params.slug)
  return {
    title: article.metaTitle,
    description: article.metaDescription,
    openGraph: {
      title: article.metaTitle,
      description: article.ogDescription,
      type: 'article',
    },
  }
}
```
